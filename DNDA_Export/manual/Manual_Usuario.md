# Manual de Usuario
**Observatorio de Prácticas Educativas — Paz, Memoria y Derechos Humanos**

## 1. Introducción
Este manual de usuario proporciona las instrucciones necesarias para interactuar y administrar la plataforma del "Observatorio de Prácticas Educativas". El sistema está diseñado para ser operado a través de cualquier navegador web moderno.

## 2. Requisitos del Sistema
- Navegador web actualizado (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari).
- Conexión a internet.
- Credenciales de acceso proporcionadas por el administrador del sistema.

## 3. Acceso al Sistema (Login)
Para ingresar a la plataforma:
1. Abra el navegador web y diríjase a la URL oficial del Observatorio.
2. Ingrese su usuario y contraseña en el formulario de acceso.
3. Haga clic en el botón "Iniciar Sesión".

## 4. Descripción de la Interfaz Principal
Una vez dentro del sistema, encontrará un menú lateral izquierdo de navegación y un área de contenido central. El menú contiene los siguientes módulos:
- Dashboard
- Conflictos y Políticas
- Tabla de Datos
- Chat con IA
- Cargar Archivo
- Mapa de Calor
- Histórico

`[INSERTAR CAPTURA DE PANTALLA: Vista general del sistema (Dashboard)]`

## 5. Módulo: Dashboard
El Dashboard es la pantalla inicial. Aquí podrá observar:
- Un resumen numérico de las prácticas registradas.
- Gráficos y estadísticas generales sobre la distribución y el estado actual de las prácticas.
Esta vista es de solo lectura y se actualiza automáticamente conforme se ingresan nuevos datos.

`[INSERTAR CAPTURA DE PANTALLA: Dashboard]`

## 6. Módulo: Conflictos y Políticas
En esta sección podrá visualizar información documental e indicadores relacionados con conflictos armados y políticas públicas integradas en las prácticas educativas.
1. Haga clic en "Conflictos y Políticas" en el menú lateral.
2. Navegue a través de la información y utilice los filtros si están disponibles.

`[INSERTAR CAPTURA DE PANTALLA: Módulo de Conflictos y Políticas]`

## 7. Módulo: Tabla de Datos (Gestión de Prácticas)
Aquí se encuentra el listado detallado de todas las prácticas.
1. Podrá buscar registros específicos utilizando la barra de búsqueda.
2. Puede filtrar la información por columnas (ej. departamento, tipo de práctica).
3. Puede consultar los detalles completos de una práctica haciendo clic sobre ella.

`[INSERTAR CAPTURA DE PANTALLA: Tabla de Datos]`

## 8. Módulo: Chat con IA
El sistema cuenta con un asistente inteligente para consultar datos.
1. Seleccione "Chat con IA".
2. Escriba su consulta en la caja de texto inferior (ej. "¿Cuántas prácticas hay en el departamento de Antioquia?").
3. El sistema le responderá basándose en la base de datos actual.

`[INSERTAR CAPTURA DE PANTALLA: Chat con IA en uso]`

## 9. Módulo: Cargar Archivo (Ingreso masivo)
Este módulo permite alimentar la base de datos con nuevos registros.
1. Vaya a "Cargar Archivo".
2. Seleccione o arrastre un archivo en formato Excel (.xlsx) o CSV con el formato requerido.
3. El sistema validará el documento y mostrará el estado de la subida. Si hay errores, se le notificará qué filas tienen problemas.

## 10. Módulo: Mapa de Calor
Visualización geográfica interactiva.
1. Al acceder al "Mapa de Calor", verá un mapa de Colombia.
2. Las áreas más iluminadas indican una mayor concentración de prácticas educativas registradas.
3. Puede acercar o alejar el mapa para ver detalles por zona.

1. Permite auditar qué usuario subió información, cuándo lo hizo y si fue exitoso.
2. Ayuda a mantener la trazabilidad de la información del Observatorio.

## 12. Cierre de Sesión
Para finalizar su trabajo de forma segura:
1. En la parte inferior del menú lateral izquierdo, ubique el botón rojo "Cerrar Sesión".
2. Haga clic sobre él; el sistema lo retornará a la pantalla de Login.
